## Parking Lot Block

Append this block to any Step 1 state display when parked items exist. Remove it once all items are flushed in Phase C.

```
┌──────────────────────────────────────────────────────────────┐
│  Parking Lot（暂存项，Phase 结束前归位）                        │
├──────────────────────────────────────────────────────────────┤
│  · [信息摘要 1] → [建议归入字段，如 Constraints]               │
│  · [信息摘要 2] → [建议归入字段，如 Non-Goals]                 │
│  · [信息摘要 3] → [待评估，可能影响 Capability/Feature]         │
└──────────────────────────────────────────────────────────────┘
```

When Parking Lot is empty, don't display this block.

## Parking Lot Flush (Phase C transition)

Display before Phase C synthesis to confirm item routing:

```
┌──────────────────────────────────────────────────────────────┐
│  Parking Lot 清理                                             │
├──────────────────────────────────────────────────────────────┤
│  ✓ [项目 1] → Vision.Constraints                             │
│  ✓ [项目 2] → Vision.Non-Goals                               │
│  → [项目 3] → 留作 Capability 阶段参考（不丢弃）               │
│  ✗ [项目 4] → 超出范围，丢弃                                  │
├──────────────────────────────────────────────────────────────┤
│  确认处理方式？                                               │
└──────────────────────────────────────────────────────────────┘
```
